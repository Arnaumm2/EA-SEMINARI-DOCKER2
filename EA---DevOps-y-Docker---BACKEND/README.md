## Descripción

Este proyecto consiste en el desarrollo del Backend EA utilizando Node.js, con una API REST implementada con Express y una base de datos MongoDB.

El Backend EA permite gestionar los recursos necesarios para la aplicación, proporcionando endpoints para la comunicación entre el cliente y el servidor.
